# Ocean Breeze Blog — 总体设计文档

## 一、项目概述

Ocean Breeze 是一个带有页面宠物互动系统的个人博客，核心功能包括：博客笔记、照片墙、桌面宠物。整体风格偏暗色系、粒子特效丰富，支持多主题切换。

## 二、当前已实现功能

### 2.1 博客系统
- CRUD 文章（标题、分类、标签、摘要）
- 标签云 + 分类筛选
- 全文搜索
- 左侧分录抽屉（可收缩）
- 右侧栏：搜索、标签云、主题切换、统计、分类管理、数据同步

### 2.2 照片墙
- 上传照片（本地文件 + URL 链接）
- 照片分类 + 标签
- 编辑/删除照片
- 灯箱大图查看
- 搜索筛选

### 2.3 主题系统
- 5 个普通主题（落日微醺、森林物语、海风微澜、浩瀚深空、雪落无声）
- 1 个里模式主题（炼狱熔岩，密码解锁）
- 每个主题有独立的粒子特效（气泡、叶片、流星、雪花、火焰）
- 深色/浅色模式切换

### 2.4 数据同步
- GitHub Gist 云端存储
- 跨设备同步
- 导出/导入 JSON

### 2.5 其他
- 座右铭打字机动画
- 阅读进度条
- Emoji 分类图标
- 页面宠物（1 只像素猫）

## 四、宠物系统详细设计

### 4.1 三只猫的角色设定

| 名字 | 形象 | 性格 | 行为特点 |
|------|------|------|----------|
| **小橘** | 橘猫 | 活泼好动 | 喜欢在页面上跑动、追逐光标、扒拉文章卡片 |
| **雪球** | 白猫 | 安静优雅 | 喜欢趴在侧边栏顶部、偶尔伸懒腰、舔毛 |
| **墨墨** | 黑猫 | 神秘调皮 | 喜欢躲在暗处、突然窜出、和粒子特效玩耍 |

### 4.2 互动系统

**左键点击（轻互动）：**
- 小橘：跳一下，发出 "喵~" 文字气泡
- 雪球：翻身露出肚皮，冒出 💤
- 墨墨：眼睛发光，短暂消失再出现

**右键菜单：**
```
┌──────────────┐
│ 📋 猫咪介绍   │  → 进入该猫的独立介绍页
│ 🎾 玩耍      │  → 猫开始追逐光标/粒子
│ 💤 休息      │  → 猫进入睡眠状态
│ 📍 召回      │  → 猫回到默认位置
└──────────────┘
```

**猫咪介绍页（新页面）：**
- 猫咪照片/形象展示
- 名字、品种、年龄、性格
- 与主人的小故事
- 喜欢的食物、玩具
- 回到主页按钮

### 4.3 三猫互动规则

- **追逐游戏**：小橘偶尔会追墨墨，墨墨会逃跑
- **一起睡觉**：雪球和墨墨偶尔会在页面角落依偎
- **争宠**：两只猫同时靠近光标时，会互相推搡
- **群体晒太阳**：浅色模式下，三只猫会聚集在页面明亮处

### 4.4 与主题的互动

- **落日微醺**：猫喜欢趴在暖色区域
- **雪落无声**：猫会试图抓雪花粒子
- **炼狱熔岩**：猫远离火焰区域，躲在角落
- **浩瀚深空**：墨墨会追逐流星

### 4.5 技术方案

- 用 Canvas 层绘制宠物（在现有粒子 canvas 之上）
- 每只猫独立的状态机（idle/walk/play/sleep/chase）
- 行为树：感知层（光标位置、粒子位置、其他猫位置）→ 决策层（选择行动）→ 行动层（执行动画）
- 右键菜单用自定义 contextmenu 组件
